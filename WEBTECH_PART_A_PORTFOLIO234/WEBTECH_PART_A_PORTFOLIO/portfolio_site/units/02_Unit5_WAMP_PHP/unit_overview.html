Unit 5 – Local Server Setup and Multi-Page Websites (WAMP/PHP)

ServerWAMPPHP

What this unit demonstrates

This unit evidences local server configuration and deployment of a basic multi-page PHP site under localhost.

Important: PHP must be run via WAMP (or equivalent). Opening .php files directly in a browser will not execute server-side code.

How to run

  1. Read: README_RUN_IN_WAMP.txt
  2. Copy site/ into your WAMP www folder
  3. Open http://localhost/<foldername>/

Source code entry: site/index.php
