Unit 10 Task 1 – Responsive Navigation

Demonstrates: responsive menu behaviour, a dropdown pattern, and smooth-scrolling anchors.

Key Features

  • Responsive menu: collapses on small screens and toggles with a Menu button.
  • Dropdown menu: opens sub-links (“More”).
  • Smooth scrolling: anchor navigation uses browser smooth-scroll.

Accessibility Notes

  • Buttons include aria-expanded states.
  • Keyboard navigation is supported via standard focus behaviour.
  • Links use clear text labels.
